현재 클라우드 4가지 기능(Feature)

  1. 대규모 (Massive Scale)
  2. 온디맨드 접근 (On-demand Access):
    종량제, 고정비 투자 없음.
  3. 데이터 집약적 특성 (Data-intensive Nature):
    MB → GB → TB → PB → XB
  4. 클라우드 프로그래밍 패러다임:
    MapReduce/Hadoop, NoSQL, Open Source, ...
