01. 나노 편집기 (Nano Editor)

GNU Nano

GNU 나노 편집기는 유닉스의 기본 편집기인 vi나 emacs 같은 처음 진입장벽이 높은 하지만 강력한 편집기보다는 처음 유닉스를 접하는 사용자가 쉽게 사용할 수 있는 편집기로 GNU에서 기존 전자우편 편집기인 Pico를 바탕으로 만들어진 편집기로 초보자에게는 정말 적합하다.

GNU Nano 편집기는 http://www.nano-editor.org/ 에서 다운로드 받을 수 있다.

(GNU Nano) 편집기 기능

Basic Features

  • 편집문서 생성, 열기, 저장
  • 이동(Navigation)
  • 찾기, 바꾸기 (Find and Replace)
  • 잘라내기, 복사하기, 붙여넣기 (Cut, Copy, Paste)
  • 텍스트 양식 (Text Formatting)
  • 실행취소, 재실생 (Undo and redo)
  • UTF-8 지원
  • 구문 강조 (Syntax highlighting)

(GNU Nano) 편집기 기능

편집문서 생성, 열기, 저장

  • 편집기 상단: 프로그램 이름, 버젼, 작업 파일 정보
  • 중간 본문: 텍스트 파일 내용
  • 하단에서 3번째줄: 시스템 메시지
  • 편집기 하단: 단축키

(GNU Nano) 편집기 기능

단축기 및 도움말

  • ^G : 도움말
  • ^X : 끝내기
  • ^O : 저장하기
  • ^W : 찾기
  • ^K : 잘라내기
  • ^U : 붙여넣기
  • ^C : 현재 위치 정보
  • ...

(GNU Nano) 편집기 기능

이동 (Navigation)

  • Ctrl+F, Ctrl+B : 왼쪽, 오른쪽 화살표
  • Ctrl+P, Ctrl+N : 위쪽, 아래쪽 화살표
  • Ctrl+A, Ctrl+E : 행의 처음으로 이동, 마지막으로 이동
  • Ctrl+V, Ctrl+Y : 다음 페이지, 이전 페이지
  • Ctrl+Space, Alt+Space : 한번에 단어 앞 뒤로 이동
  • Ctrl+_+숫자+,+숫자 : 바로 이동
  • Ctrl+C : 현재 커서 위치 정보
