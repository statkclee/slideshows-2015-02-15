Hello, and welcome to the Python section of the Software Carpentry bootcamp.

파이썬(Python)은 ...

  • 대중적이며(popular),
  • 강력하며(powerful),
  • 배우기 쉬운(easy to learn)
  • 프로그래밍 언어인데(programming language),
  • 가독성이 좋고(human readable),
  • 구분자가 공백이고(whitespace delimited),
  • 무료이고(free),
  • 공개 소프트웨어다(open source).
So what is Python anyway? Python is a programming language which is used to: - develop analysis pipelines - process and analyse data - visualize data, and generate publication-quality graphics - develop new statistical methods It is also a free open-source project, making it completely free to download and extend.

파이썬은 정말 훌륭한 라이브러리가 많다.

  • 수학(mathematics)
  • 그래프 그리기(plotting)
  • 과학 영역(domain science)

파이썬으로 할 수 있는 작업.

  • 임의 파일 형식을 다른 파일 형식으로 데이터 파일 변환
  • 이미지 파일을 분석
  • 디렉토리에 있는 모든 파일을 루프를 돌려, 분석을 하고, 분석결과를 그래프화한다.
  • 거의 모든 것을 할 수 있다.
